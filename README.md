# User Management API

## Prerequisites

Ensure you have the following installed:
- Python 3.0+
- pip

## Installation

1. Clone the repo
    >git clone https://github.com/yourusername/user_management.git
    >cd user_management

2. Install dependencies

3. Import user data
    >python scripts/import_users.py
  
5. Run the Flask app:
    >python run.py
